# 📊 Automação de Relatórios de Sinistros - CET (São Paulo)

Este projeto tem como objetivo automatizar o processamento e a inserção de dados de sinistros para a Companhia de Engenharia de Tráfego (CET). A solução substitui o preenchimento manual de planilhas de valores, garantindo maior integridade dos dados e agilidade no fluxo de trabalho.

## O Projeto

A automação foi desenvolvida para realizar a extração e o tratamento de dados, inserindo valores automaticamente em tabelas do **Excel**. Além da automação de back-end, o projeto conta com um **Dashboard Front-end** para visualização centralizada das ocorrências.

### Principais Funcionalidades:
- **Automação ETL:** Processamento de dados de sinistros e inserção automática em arquivos `.xlsx`.
- **Filtro Geográfico:** Foco exclusivo em dados da cidade de **São Paulo**.
- **Interface Web:** Front-end interativo para exibição de indicadores e informações detalhadas.
- **Validação de Dados:** Sistema de checagem para evitar duplicidade ou erros de inserção na tabela.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python (Pandas/Openpyxl para manipulação de Excel)
- **Front-end:** React.js / JavaScript / HTML / CSS
- **Versionamento:** Git / GitHub
